# Guide du backoffice — Site Dr Mohamed Moussa

Ce guide explique comment modifier le site **sans toucher au code**, depuis l'espace d'administration.

## Se connecter

1. Aller sur **https://drmohamedmoussa.com/admin/**
2. Cliquer sur **« Se connecter avec GitHub »** et autoriser avec le compte du cabinet.
3. Vous arrivez sur le tableau de bord.

À chaque **Publish**, le site se met à jour automatiquement en 1 à 2 minutes.

## Ce que vous pouvez modifier

Le menu de gauche est organisé en rubriques :

| Rubrique | Contenu modifiable |
|----------|--------------------|
| **Réglages du cabinet** | Coordonnées, téléphones, e-mail, adresse, horaires, liens Google/Med.tn, nombre d'avis |
| **Page d'accueil** | Tous les textes et **toutes les photos** de l'accueil, + les tuiles « Découvrir » |
| **Pages du site** | Photos **et** textes de chaque page : Le médecin, Le cabinet, Chirurgie, Notre matériel, Activité scientifique, Contact, Prendre rendez-vous, Blog, page locale |
| **Services (13 pages)** | Chaque service : textes, symptômes, traitements, FAQ + une photo de bandeau |
| **Articles de blog** | Créer / modifier / supprimer des articles (avec photos dans le texte) |
| **Avis Google affichés** | Les avis montrés sur l'accueil |
| **Congrès et communications** | Les photos et textes des congrès |

## Photos : ajouter, remplacer, supprimer

- **Remplacer une photo** : cliquer sur le champ photo → **Choose an image** → téléverser une nouvelle image ou en choisir une existante.
- **Galeries** (cabinet, bloc opératoire) : chaque photo est une ligne. Bouton **Add** pour ajouter, l'icône corbeille pour supprimer, glisser-déposer pour réordonner.
- Les photos sont automatiquement optimisées (chargement rapide). Pensez à remplir la **description** de chaque photo (utile pour Google et l'accessibilité).

## Mettre un mot en gras

Dans les zones de texte qui l'indiquent, entourez le mot de deux étoiles :
`un cabinet **moderne**` s'affiche : un cabinet **moderne**.

## Bon à savoir

- Le **logo** et l'identité visuelle ne sont pas modifiables depuis le backoffice (protégés).
- En cas de doute, ne supprimez pas une rubrique entière : modifiez seulement le texte ou la photo.
- Chaque modification est enregistrée dans l'historique : rien n'est jamais définitivement perdu.
